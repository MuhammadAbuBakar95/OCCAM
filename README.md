Introduction
============




Building and Installing
=======================

```
make
make install
```

Using OCCAM
===========

You can choose to record logs from the OCCAM 
tool by setting the following variables:

```
  export OCCAM_LOGFILE={absolute path to log location}
  export OCCAM_LOGLEVEL={INFO, WARNING, or ERROR}
```
